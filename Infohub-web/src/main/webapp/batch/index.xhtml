Batch update from Excels

Batch update from contact excel file.

Uses:

Create new contacts for any partner in Infohub. Only new contacts found in excel will be processed if the partner is found in Infohub.This will not delete or update any existing contact from Infohub.

Requirement:

This process requires 1 partner.xls downloaded from Infohub partner list page, and process the contact tab only.

Batch update from SAP excel file.

Uses:

Create or update partner in Infohub with the technical and CLM contacts found in the excel. Only create new technical and CLM if it doesn't exist in infohub.

Requirement:

This process requires 2 inbound excel downloaded from SAP PIF and PC1 system, 2 outbound excel downloaded from SAP PIF and PC1 system, and 1 CLM excel downloaded form SAP PIF.
